📋 Задания

Задание 1.1. В ~/education/typography/index.html создать уникальную тематическую страницу, содержащую: заголовки, параграфы, индексы, степени, списки (вложенные ненумерованные и нумерованные), горизонтальную линию. Все стили — только через CSS, без устаревших атрибутов.

Задание 1.2. В passport.html создать страницу с данными паспорта (страницы 2 и 3). Личные данные (кроме фамилии, имени и года рождения) можно изменить. Получить данные из DOM: document.getElementsByTagName('p')[2].childNodes[1].innerText

Задание 1.3. В passport_find_edit.html создать страницу, изменяющую данные паспорта по событию (кирилличные → латинские символы).
